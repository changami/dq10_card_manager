dq10_card_manager
====

チームやグループで、カードの枚数や残りのカード時間を共有することができます。

## Description

1. 共有したいチームやグループを作成します。
1. 作成したチームやグループでログインし、各自カード情報を入力します。
1. 情報が入力されると残り時間が少ない順に表示されるので、それを参考に遊んでください。

何かあればPull Requestでも[連絡](https://twitter.com/chan_gami)でもいただけると幸いです。
